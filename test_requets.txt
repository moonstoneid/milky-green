Start authorization
-------------------
(scopes: openid, profile, email)
http://127.0.0.1:9000/oauth2/authorize?client_id=test&response_type=code&scope=openid%20profile%20email&state=12345&redirect_uri=http://127.0.0.1:8080/authorized

Complete authorization
----------------------
curl -X POST \
  http://127.0.0.1:9000/oauth2/token \
  -H 'Authorization: Basic dGVzdDp0ZXN0' \
  -H 'Content-Type: application/x-www-form-urlencoded' \
  -d 'grant_type=authorization_code&code=[CODE]&redirect_uri=http%3A%2F%2F127.0.0.1%3A8080%2Fauthorized'

Query user information
----------------------
curl -X GET \
  http://127.0.0.1:9000/userinfo \
  -H 'Authorization: Bearer [ACCESS_TOKEN]'
